## Project Description

The Online Examination Management System is a web-based application developed using Django Framework.

The system allows an administrator to manage:

- Students
- Examinations
- Results

---

## Objectives

- Register students
- Create examinations
- Record examination results
- Manage examination records

---

- Django

- Bootstrap

## Installation

Clone the project

```bash
git clone repository_url
```

Install requirements

```bash
pip install -r requirements.txt
```

Apply migrations

```bash
python manage.py migrate
```

Create admin

```bash
python manage.py createsuperuser
```

Run project

```bash
python manage.py runserver
```

Open browser

```
http://127.0.0.1:8000/
```

Admin Dashboard

```
http://127.0.0.1:8000/admin/
```
